# Startpage

Custom Startpage

### Build Instructions
+ `npm install`
+ `grunt build`
+ Static files are now in build

### Testing
+ `grunt watch` to build when files change
+ View webpage:
    - Open `build/index.html` in a browser
        - Or run a server, `cd build && python -m http.server`
        - Navigate to [localhost:8000](localhost:8000)
